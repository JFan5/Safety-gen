## 难度分析

问题的难度主要由以下因素决定：

1. **网格大小 (X × Y)**: 网格越大，搜索空间越大，问题越难
2. **形状数量 (shapes)**: 形状越多，需要匹配的锁-钥匙对越多
3. **钥匙数量 (keys)**: 钥匙越多，需要管理的对象越多
4. **锁数量 (locks)**: 锁越多，需要解锁的位置越多

通常，网格大小是影响难度的主要因素，其次是形状、钥匙和锁的数量。
