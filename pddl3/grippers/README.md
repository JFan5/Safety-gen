### 难度分析
问题的难度主要由以下因素决定：
1. **机器人数量 (n)**: 机器人越多，协调越复杂
2. **房间数量 (r)**: 房间越多，搜索空间越大
3. **物体数量 (o)**: 物体越多，需要搬运的任务越多
